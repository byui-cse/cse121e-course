CSE 121e: Week 02

Ponder and Prove

Introduction

You work for a TV program. The Show is one that has competition between teams and cooperation within teams.

Your Task

Your boss needs help selecting good teams from persons that have been selected to be potential participants.

  1. Create a list containing 20 persons of each gender. Each element of the list must be a tuple that looks like this, {Propensity, Name, gender_atom}. The propensity to do well in the game is numeric with a scale of your choice. This list will be the data as if they had been pulled from a database.

    Hints: 1)Use list comprehensions to generate the list. 2)Depending on how you choose to design and write this function, you may need other BIF's described in the list API found in this week's reading.

  2. Use the shuffle function in the provided list_lib to scramble the list.

  3. Write a function that creates teams of size 5 where each element of the list is a tuple following this format. {total_propensity,[{name,gender},{name,gender},{name,gender},{name,gender}]}

Review

After completing the previous steps:

  1. Review one of the many possible solutions for this task,
  2. Compare and contrast your code with the possible solutions’ code, and
  3. Make note of any improvements you could make for future tasks.
