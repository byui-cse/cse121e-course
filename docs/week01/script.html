WSL and Linux Users

Install rebar3 by entering this in your terminal application sudo apt update sudo apt install rebar3

macOS Users

Install HomeBrew by opening the terminal application and entering, /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" Then install rebar3 by entering this in your terminal brew install rebar3

Getting Started

As part of your team, interact with your Erlang Learning Assistant to review Python and see how what you learned there relates to Erlang. Discuss each item it brings up until everyone in your team is secure and confident in the basic programming principles it brings up. If anyone in your team doesn't understand something, make sure you ask your assitant for more information about the point it brought up.

Running Erlang Code

In your terminal, enter this rebar3 new lib first then 'cd' into the 'first' directory that was just created. Create a file called silly.erl using any text editor you want, VSCode is fine if you already have it. Make sure the file is created in the first/src directory. All of your source code for a project goes in the src(source) directory of the project. Put this code in it and save the file. 

 -module(silly).
 -export([sum/2]).

 sum(X,Y)->X+Y.
Enter this into your terminal rebar3 shell Your code is now ready to run. Enter this in the Erlang shell. Enter this into your terminal silly:sum(2,4). You should see the number 6. Congratulations! You are all set up and ready to go.
