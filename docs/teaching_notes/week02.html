Week 02 Goals

  1. Introduce Erlang syntax for basic concepts such as operators and variables.
  2. Introduce the new-to-the-students concept of immutability.
  3. Introduce the new-to-the-students concepts of atoms and tuples.
  4. Introduce the students to how Erlang lists work, and why they work this way.
  5. Introduce the students to the Erlang list BIFs.

Week 03 Overview

For next week, we'll be learning about how to create functions in Erlang. Functions in Erlang are very different from Python. As you go through the reading material, make sure you try each of the examples. You are encouraged to look for other resources online as you encounter questions. Here are a couple of resources you might find helpful:

  • https://erlang.org/ - This site is the official Erlang website, with documentation and examples.

  • https://learnyousomeerlang.com - This is a great online book that is free. It contains examples that might help you further understand Erlang. This is an optional resource, but great to use if you would like to see additional examples or read more about specific topics in Erlang.

As a reminder, you can create a project using rebar3 by running the following command:

rebar3 new lib <name-of-project-here>

This command will create the structure for a new project, including a src folder and an Erlang file to get you started. You can include additional Erlang files in the src folder and rebar3 shell will compile all of them. Then you can use the REPL to try running your functions.
